法币支付接入流程

  1. 创建或导入钱包
    2. 要使用派酷支付,商户必须首先创建或导入钱包.
  2. 获取燃料
    3. 如果是新创建的钱包,还可以免费获取燃料.进入钱包页面,点击获取免费燃料按钮.每个新钱包只能获取一次.
  3. 注册派酷会员
    4. 进入会员页面,输入推荐码,免费加入派酷消费商生态会员系统.
  4. 注册商户
    5. 进入商家页面,输入商家名称,商家标志,地址,联系人姓名,电话,传真,电子邮件,网站,开放时间,关闭时间,业务内容,返利百分比,主币种,税率,锁定天数,商家推荐地址等信息, 点击创建按钮, 完成商家的注册过程并等待派酷官方的审核, 审核通过后即可使用.
  5. 商户信用充值
    6. 进入商家页面,点击商家信用账户页面, 在新页面中点击左上方的添加按钮, 选择要充值的币种和数量,点击确认,完成充值过程.充值前请确保交易所有足够的该币种的余额.如果余额不足,可以通过钱包的存币操作或交易所的交易功能获取足够的余额.
  6. 获取商户通证押金
    7. 进入商家页面,点击显示通证押金链接,输入钱包密码,即可获取商户通证押金, 该通证押金在下一步的调用payWithFiat接口中使用.
  7. 获取商家ID
    8. 进入商家页面,得到商家ID.
  8. 完成法币支付
    9. 通过常规的法币支付接口(信用卡,paypal,苹果支付,google支付等)完成法币支付过程.
  9. 调用payWithFiat接口完成派酷支付
    10. 法币支付确认后,调用payWithFiat接口完成派酷支付,接口输入输出数据格式为JSON。
    请求方式: POST请求
    请求url:
